# Review Instructions
## SEC Filing Q&A Data Collection

**Date:** April 2026

---

## Your Task

You have claimed a review task. You will receive a completed annotation task set — 10 submitted Q&A pairs — and must assign a rating to each submission and implement any corrections.

You will receive all documents referenced in the submission, including any filings beyond the annotator's three assigned companies.

Payment is based on time at $20/hour. Category B and Category C reviews earn a per-question bonus on top of base pay.

---

## Rating Scale

For each submission, assign a single rating:

| Rating | Meaning |
|--------|---------|
| 2 — Accept | Factually accurate, self-contained, correct difficulty level, no PII |
| 1 — Revise | Minor fixable issue; implement correction directly in the record using bold text |
| 0 — Reject | Factually wrong, not self-contained, or missing source |

---

## Review Checklist

Check each Q&A against the following:

- **Self-containedness:** does the Supporting Facts section meet the standard for its category?
  - *Easy:* source citation with page number present
  - *Medium:* all calculation inputs provided with page numbers
  - *Hard:* each company-level claim has a cited fact or quoted language with filing and page; absent disclosures documented with where the reviewer looked

- **Accuracy:**
  - *Easy:* open the cited page and verify the fact is stated in the filing as the answer claims
  - *Medium:* independently run the calculations using the Supporting Facts values; flag if the result doesn't match the answer, or if a metric is misidentified (e.g., gross profit used instead of operating income)
  - *Hard:* confirm all cited filings are real and from October 2023 or newer; evaluate whether the overall synthesis is logically supported by the evidence cited; flag if an absent disclosure is not documented, a conclusion overstates what the evidence supports, or a causal/comparative claim is not logically justified

- **Correct difficulty level:** does the Q&A match Easy / Medium / Hard as labeled?

- **PII:** flag and anonymize any personal names, contact information, or identifying details.

---

## Revising the Submission

If you assign a rating of 1, you will need to do the following. First, explain exactly why the Q&A is insufficient — is the Supporting Facts section incomplete? Is a critical detail missing? Is the reasoning logically flawed? Be specific about what the problem is before proposing any fix.

Second, suggest a correction that would, in your view, elevate the item to a 2. Then implement that correction directly in the record. Use **bold text** for every change you make, so the project lead and original annotator can see exactly what you altered.
