# README: 3_annotator_data/

---

## Purpose

This folder holds the assignment packets distributed to annotators at the start of Phase 2, and is where completed submissions are returned. Each CSV is pre-filled with everything an annotator needs to know about their assignment — their companies, filing types, dates, SEC search links, and suggested PDF filenames. Annotators fill in their questions, answers, and source details, then return their completed CSV alongside the PDF files they downloaded.

Each CSV has 10 rows (3 Category A + 3 Category B + 4 Category C). The following fields are pre-filled: `annotator_id`, `sector`, `question_category`, and all source document metadata (company, filing type, filing date, SEC URL, suggested PDF filename). The following fields are left for the annotator to complete: `question`, `answer`, `additional_info`, confirmed `pdf_filename`, and `source_4_plus_notes` for any Category C sources beyond the 3 assigned companies.

Annotators return their completed CSV and downloaded PDFs to this folder.
